# Sensor Reference Files

This folder stores local reference material for the non-contact liquid level sensor used in this project.

## Files

- `xkc_y25_connection_schematic.png` - connection/wiring schematic reference
- `xkc_y25_pinout_board.png` - pinout and interface board reference

## Source

- DFRobot Wiki: https://wiki.dfrobot.com/Non_Contact_Capacitive_Liquid_Level_Sensor_SKU_SEN0368
