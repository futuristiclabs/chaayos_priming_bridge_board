# chaayos_priming_bridge_board

We are creating a bridge board where four sensors, which are priming detection sensors. These are fit to pipes, which are RO and PDIP pipes, which kind of detect if the liquid is passing through the pipe or not. These sensors come in a four-pin layout, which is a signal, ground, power, and also a toggle. Our main board, the Chaayos board, the input has to be through I2C. What we're trying to do is to build a bridge board in which all these four sensors are connected to the main board via an I2C line.
